# Student Performance Dashboard (Mini EDA)

This project presents a mini data visualization dashboard using the Student Performance dataset. 
The goal is to analyze student performance and communicate insights using clear and effective visualizations.

##  Project Overview
This task focuses on creating a dashboard-style analysis by combining data preparation, feature engineering, and multiple visualizations.

##  Data Preparation
- Checked dataset for missing values
- No missing values were found
- Prepared dataset for analysis

##  Feature Engineering
- Created a new column **Result** (Pass/Fail based on final grade)
- Created **GradeCategory** (Low, Medium, High)

##  Dashboard Visualizations
- Histogram of final grades
- Bar chart of average grade by gender
- Boxplot of study time vs grades
- Scatter plot of study time vs grades
- Correlation heatmap

##  Key Insights
- Students who study more tend to perform better
- Previous grades strongly influence final performance
- Gender shows slight variation in performance
- Most students fall in the average performance category

##  Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📂 Files Included
- task4_student_dashboard.ipynb (Jupyter Notebook)

## 🎯 Conclusion
The dashboard helps in understanding patterns in student performance and highlights important factors such as study time and previous grades. 
Visualizations make it easier to interpret data and communicate insights effectively.
